Schéma réseau ⬇

Description

Internet : Connecté au pare-feu pfSense.

pfSense Firewall : Point central du réseau qui connecte tous les autres composants :

    Web Server (Apache) : Serveur web hébergé, connecté au pare-feu pfSense.

    Backup Web Server : Serveur de sauvegarde du site web, connecté au pare-feu pfSense.

    Windows Client : Client Windows connecté au pare-feu pfSense.

    Linux Client : Client Linux connecté au pare-feu pfSense.

Sauvegarde et restauration fonctionnelle automatique (des fichiers sources du site web) ⬇

Mettre en place Cron pour faire que la sauvergarde se fasse chaque jours a minuit.

avec la commande 0 0 * * * /usr/local/bin/backup.sh

Une documentation pour la restauration ⬇

Afin de créé la sauvergarde/backup du site web il faut :

1. Crée le fichier qui sera partager entre le serveur web et le serveur de sauvegarde.

2. Installer le fichier sur chaque serveur web et sauvegarde

3. Crée un script afin de configurer le backup et l'executable

4. Checker si la sauvegarde a bien été effectuer.

avec la commande : sudo rsync -avz --delete /var/www/html /mnt/shared-backup

Il faut run sync manuellement pour verifier la sauvegarde
